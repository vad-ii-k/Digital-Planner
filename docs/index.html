Digital Planner for PDF annotation apps

It includes dated yearly, monthly, weekly, daily and other planning pages interlinked with over 20000 hyperlinks, enabling you quick access to any related section in only one click!

2023 Calendar

Contains links to any month, week and day of the year. You can return to it using the 1st button of the lower navigation menu.

Monthly Calendar

Contains links to any week and day of the month, as well as to the previous and next months.

Weekly Planner

Contains links to any day of the week, as well as to the previous and next weeks.

Daily Planner

Contains links to any week and day of the month, as well as an additional link to the incoming tasks page (Inbox).

Inbox

A page for postponed tasks. You can switch to it by clicking the 2nd menu button.

Projects Overview

Contains links to the project details pages (Project Planner). You can open this template by clicking the 3rd menu button.

Project Planner

Contains buttons for switching between projects.

Habits Tracker

Contains links to any day of the month, as well as buttons for switching between the habits trackers of the previous and next months.

Switching to this template is carried out using the 4th menu button, the month of the tracker is determined depending on the page opened at the time of clicking.
